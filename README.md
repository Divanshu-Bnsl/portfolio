# Divanshu Bansal Portfolio

A single-page personal portfolio built with HTML, CSS, and JavaScript. It presents my background as a software developer and AI/ML engineer, along with featured projects, technical skills, and contact details in a polished interactive layout.

## Features

- Full-screen hero section with a typewriter-style role animation
- Animated aurora-style background and cursor glow effect
- Motion controls for subtle, balanced, and cinematic experiences
- Sections for About, Skills, Projects, and Contact
- Responsive layout designed to work well on desktop and mobile
- Interactive contact form UI
- External links to GitHub projects and profile

## Sections

- **Home** - intro banner with call-to-action buttons
- **About** - short bio and summary stats
- **Skills** - programming languages, tools, and specialties
- **Projects** - featured projects including Indexify, SavoriaRMS, RoleMatch-AI, and Ascend Quest
- **Contact** - email, GitHub link, and message form

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts
- SVG icons and canvas-based visual effects

## How To Use

1. Download or clone the repository.
2. Open `portfolio.html` in a browser.
3. Edit the content inside the file to update your name, links, projects, and contact details.
4. If you want to publish it with GitHub Pages, rename the file to `index.html` or set the repository homepage accordingly.

## Customization Notes

- Update the email and GitHub profile links in the Contact section.
- Replace the featured projects with your own project cards.
- Adjust colors, fonts, and motion settings in the CSS variables near the top of the file.
- The contact form currently shows a placeholder alert and does not submit to a backend service yet.
